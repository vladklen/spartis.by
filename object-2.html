Строительство двухквартирного блокированного жилого дома вместо сносимого по адресу г.Минск пер.Ярошевиский, 11

Заказчик: Частное лицо

Генподрядчик: ООО “СПАРТИС”

С 2020 года по настоящее время компания ООО «СПАРТИС» в качестве генеральной подрядной организации выполнялет комплекс работ по строительству объекта «Строительство двухквартирного блокированного жилого дома вместо сносимого по адресу г.Минск пер.Ярошевичский 11». ООО «СПАРТИС» выполняет работы по устройству котлована, монолитного каркаса здания (фундаменты, перекрытия, лестницы, стены, колонны), каменные работы (устройству наружных стен и внутренних перегородок), работы по устройству кровель , комплекс отделочных работ, работы по благоустройству, сантехнические и электромонтажные работы. Особенностью данного проекта явлется высочайщие требования к качеству бетонных работы. Для устройста перекрытия использовалась только новая фанера, что позволило получить поверхности класса А3. Высота перекрытий на данном объекте от 3,7 до 8,5 м. Для устройство монолитного каркаса в помещениях со вторым светом высотой до 8,5м применялась немецкая опалубная система Hunnebeck ID 15.
