Торговый центр по ул.Тимирязева

Заказчик: СООО «БелМартПлаза»

Подрядчик: ООО «СПАРТИС»

С 2014года по 2015 год компания ООО «СПАРТИС» участвовала в строительстве объекта «Торговый центр по ул.Тимирязева» в г.Минске. ООО «СПАРТИС» выполняло земляные работы, работы по устройству монолитного каркаса (колонны, стены, перекрытия, въездная рампа), каменные работы по устройству кладки наружных стен и монтаж металлоконструкций (стойки фахверка). Уникальностью данного объекта является: пролет между несущими колоннами (12м.), утолщенное перекрытие (до 600мм) и высота этажа (до 6,4 м). Для реализации данных конструктивных особенностей потребовалось применение усиленных немецких опалубочных систем ID 15 Hunnebeck которые были приобретены специально для данного объекта.
